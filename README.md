# 決断診断アプリ - 承継・M&Aの決断参謀

## 概要
「息子に継がせるか、M&Aで売るか」という経営者の悩みを20分で整理する診断アプリです。

## 機能

### 1. ランディングページ
- サービス紹介
- ベネフィット説明
- 診断の流れ
- CTA（無料診断開始）

### 2. 診断フォーム（7つの質問）
1. 今、一番悩んでいることは？
2. 後継者候補はいますか？
3. 理想の引退時期は？
4. 会社を残す上で、一番大事にしたいことは？
5. 今、一番の不安は？
6. これまでに誰かに相談しましたか？
7. もし今日、決断できるなら、どうしたいですか？

### 3. 診断結果（4象限マトリクス）
- **A象限（親族承継・準備OK型）**: 後継者準備度高 × 想いの強さ高
- **B象限（親族承継・準備不足型）**: 後継者準備度低 × 想いの強さ高
- **C象限（M&A・想い薄い型）**: 後継者準備度高 × 想いの強さ低
- **D象限（迷走中・要整理型）**: 後継者準備度低 × 想いの強さ低

### 4. 診断レポート
- 現在地の可視化
- 次の一手（3つのアクション）
- 注意すべきリスク
- CTA：無料ヒアリング予約
- レポートダウンロード機能

### 5. 管理者機能
- ログイン認証
- 診断データ一覧
- 統計情報表示
- CSVエクスポート

- ブラウザのローカルストレージを使用
- データは各ユーザーのブラウザに保存されます
- CSVエクスポートで外部保存可能

## カスタマイズ
- `js/app.js`の`ADMIN_USERNAME`と`ADMIN_PASSWORD`を変更してログイン情報を変更
- `css/style.css`でデザインをカスタマイズ
- 診断ロジックは`calculateDiagnosis()`関数で調整可能
